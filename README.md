# Game Dev Mate

Game Dev Mate (**GDM**) is a game devlopment library on C++, currently in it's first alpha version.

## Dependencies

```cd
sudo apt-get install cmake
sudo apt-get install libsfml-dev
```

## Build

```cd
mkdir build
cd build
cmake ..
make
```
